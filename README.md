## üöÄ Quick Deploy to GitHub Pages (FREE)

### Step 1: Create GitHub Account
1. Go to https://github.com
2. Sign up for free
3. Verify your email

### Step 2: Create Repository
1. Click "+" icon (top right) ‚Üí "New repository"
2. Name: `mathieudelloye.github.io`
3. Description: "Portfolio - ServiceNow ITSM & AI Product Owner"
4. Select "Public"
5. Click "Create repository"

### Step 3: Upload Files
**Option A: Using GitHub Web Interface (Easiest)**
1. Go to your new repository
2. Click "Add file" ‚Üí "Upload files"
3. Drag and drop these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `cv_mathieu_servicenow_1page.html` (optional, for CV download)
4. Click "Commit changes"

**Option B: Using Git Command Line**
```bash
git clone https://github.com/YOUR-USERNAME/mathieudelloye.github.io.git
cd mathieudelloye.github.io

# Copy the portfolio files here
# Then:
git add .
git commit -m "Initial portfolio setup"
git push origin main
```

### Step 4: Enable Pages
1. Go to Repository Settings (‚öôÔ∏è gear icon)
2. Scroll to "GitHub Pages" section
3. Select "Deploy from branch"
4. Select "main" branch
5. Click Save

### Step 5: Access Your Portfolio
Your site will be live at: `https://mathieudelloye.github.io`

**That's it! Free hosting, no credit card required.**

## üîß Customization

### Update Contact Information
Edit in `index.html`:
```html
<a href="mailto:YOUR-EMAIL@domain.com">YOUR-EMAIL@domain.com</a>
<p>+YOUR-PHONE-NUMBER</p>
```

### Change Colors
Edit in `styles.css`:
```css
:root {
    --primary-color: #1a5f7a;      /* Header blue */
    --secondary-color: #ff6b6b;    /* Accent red */
    --accent-color: #1565c0;       /* Detail blue */
}
```

### Add LinkedIn/GitHub Profile
Edit in `index.html`:
```html
<a href="https://linkedin.com/in/YOUR-USERNAME" target="_blank">LinkedIn</a>
<a href="https://github.com/YOUR-USERNAME" target="_blank">GitHub</a>
```
